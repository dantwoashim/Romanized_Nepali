# User-Submitted Data

This folder is for consented, redacted validation workflows only.

- Raw files must stay under `raw/` and are gitignored.
- Private working files must stay under `private/` and are gitignored.
- Redacted text can be prepared under `redacted/`.
- Committed fixtures under `fixtures/` must reference consent metadata.
- Consent metadata lives under `metadata/` and must match `consent.schema.json`.

Do not commit raw documents, private identifiers, signatures, phone numbers, citizenship numbers, addresses, school/government private IDs, or health/legal private details.
