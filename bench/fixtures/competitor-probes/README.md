# Competitor Probe Fixtures

These fixtures support manual black-box comparison only.

- Do not scrape competitors.
- Do not call private APIs.
- Do not copy competitor code, mappings, or generated data into Lekh.
- Competitor output fields may be filled manually after small public UI testing.
- Empty competitor outputs mean "pending manual collection".

The benchmark computes Lekh output locally and checks protected-span safety. It does not claim Lekh beats any named tool.
