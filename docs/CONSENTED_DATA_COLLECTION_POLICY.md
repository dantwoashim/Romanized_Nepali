# Consented Data Collection Policy

Lekh must not silently collect typed text.

## Allowed

- Manual user-submitted examples with explicit consent.
- Redacted bug reports.
- Local-only diagnostic summaries without raw typed text.
- Exported personal memory only when the user triggers export.

## Not Allowed

- Hidden keystroke telemetry.
- Raw document uploads without consent.
- Fabricated consent metadata.
- Competitor outputs used as ground truth.
- Real documents committed without redaction and consent.

## Required Metadata

- submitter consent checkbox
- date
- platform
- mode
- redaction confirmation
- permission scope

Real-document public claims remain blocked until this process produces approved samples.
