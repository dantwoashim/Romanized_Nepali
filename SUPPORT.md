# Support

For bugs, feature requests, and conversion failures, use GitHub issues.

Choose the most specific issue form:

- Preeti conversion failure
- Romanized typing failure
- Bug report
- Feature request

For private or sensitive examples, do not paste the original text into a public issue. Create a minimal synthetic example, or use the in-app feedback path with only content you explicitly want reviewed.

For security concerns, follow [`SECURITY.md`](SECURITY.md).
