# Lekh Assistant

Privacy-first Nepali typing assistant for desktop workflows.

Week-1 MVP:

- Preeti to Unicode converter as the primary tab.
- Beta Romanized Nepali typing with candidates.
- Local seed suggestions and basic spell hints.
- Shared Unicode normalization.
- Explicit feedback and desktop beta interest path.
- PWA shell with local browser processing.

## Commands

```bash
npm install
npm run test
npm run build
npm run verify
npm run dev
```

No typed text, converted text, dictionary queries, keystrokes, clipboard content, spell tokens, or output are sent by the app automatically.
