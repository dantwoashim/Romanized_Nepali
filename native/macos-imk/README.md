# macOS IMK Scaffold

This folder documents and sketches the macOS InputMethodKit path for Lekh Keyboard.

It is not a production macOS input method. Production requires a real IMK bundle, XPC service, macOS app testing, Developer ID signing, notarization, installer validation, and pilot feedback.
