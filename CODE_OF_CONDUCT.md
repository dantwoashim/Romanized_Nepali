# Code of Conduct

This project is for practical Nepali language tooling. Keep discussion useful, respectful, and focused on the work.

Expected behavior:

- Be direct without being hostile.
- Assume good intent, but correct mistakes clearly.
- Respect privacy, especially around real documents and user-submitted text.
- Keep language, identity, and regional preference debates grounded in fixtures, contracts, and user evidence.
- Disagree with claims and implementation choices, not with people.

Unacceptable behavior:

- harassment, threats, insults, or personal attacks
- publishing private documents or personal data
- pressuring contributors to reveal sensitive user examples
- knowingly adding unclear-licensed data
- making false claims about official authority, adoption, or validation

Maintainers may edit, close, or remove issues, comments, and pull requests that violate these expectations.
